%native_language_name%~%target_language_name%~Pause Mode

Use this mode when initially learning a lesson. In this mode you'll be repeatedly presented with:

  • A word or phrase in %native_language_name%
  • The same word or phrase translated into %target_language_name%
  • A pause during which you can repeat the %target_language_name% translation

For example:

  • "%native_language_example%"
  • "%target_language_example%"
  • Pause
