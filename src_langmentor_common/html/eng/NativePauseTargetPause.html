%native_language_name%~Pause~%target_language_name%~Pause Mode

Use this mode to practice translating from %native_language_name% into %target_language_name%. In this mode you'll be repeatedly presented with:

  • A word or phrase in %native_language_name%
  • A pause during which you can attempt to translate the %native_language_name% into %target_language_name%
  • A correct %target_language_name% translation
  • A pause during which you can repeat the %target_language_name% translation

For example:

  • "%native_language_example%"
  • Pause
  • "%target_language_example%"
  • Pause
